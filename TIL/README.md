# C203 TIL Branch
## 🔍 TIL이란? (Today I Learned)
하루 동안 배운 내용을 정리하고 기록하는 공간입니다.
각자가 학습한 내용을 개인 폴더에 업데이트하여 꾸준히 성장할 수 있도록 합시다.

## 📂 폴더 구조
```bash
📦 C203-TIL  
 ┣ 📂 개인_폴더  
 ┃ ┣ 📜 YYYY-MM-DD.md  # 날짜별 학습 내용 정리  
 ┃ ┗ 📜 README.md  # 개인 TIL 관리 가이드 (선택)  
 ┣ 📜 README.md  # TIL Branch 개요  
 ```

## ✏️ 작성 규칙
#### 1. 날짜별로 정리  
- YYYY-MM-DD.md 형식으로 파일을 생성하여 정리
#### 2. 짧고 명확하게 기록 
- 핵심 개념, 배운 점, 느낀 점 등을 요약
#### 3. 자유롭게 작성 가능  
- 코드, 그림, 표 등 다양한 형식 활용 가능

## 🚀 목표
- 꾸준한 학습 습관 형성
- 배운 내용을 체계적으로 정리하여 복습
- 서로의 학습 내용을 공유하며 성장

함께 꾸준히 학습하고 발전해 나갑시다! 💪
